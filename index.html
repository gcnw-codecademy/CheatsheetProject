Description Example Syntax
href specifies a link destination <a href="link destination">Link Text</a>
src The required src attribute specifies the URL of the image. <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
width The width attribute specifies the width of an image, in pixels. <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
height The height attribute specifies the height of an image, in pixels. <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
alt The required alt attribute specifies an alternate text for an image, if the image cannot be displayed. <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
style Setting the style of an HTML element, can be done with the style attribute. <tagname style="property:value;">
lang The lang attribute specifies the language of the element's content. < lang="fr">Ceci est un paragraphe.</p>
